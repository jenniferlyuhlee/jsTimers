Timers Exercise

In the console, call the function countDown(num) with a single argument, a number. The function will countdown from that number to 0, at which it will print "DONE!"

In the console, call the function randomGame() without any arguments. The function will generate random numbers ranging from 0 to 1, counting each time a number is generated until a number greater than .75 appears. The random selector will stop and log how many times it took to reach that number.
